## ЛР №5, ООП
Данная лабораторная работа может использоваться как образец, но:
- Необходимо также создать в базовом классе такие вещи, как:
- - Конструктор копирования
- - Оператор присваивания
- Операторы присваивания производных классов должны учитывать поля базового
- Рекомендуется явно вызывать конструктор базового класса в производных
